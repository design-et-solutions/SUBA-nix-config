```
├── colorscheme -> generate custom colorscheme based on wallpaper
├── fontProfile -> add custom font profile
│   ├── packages
│   │   ├── ${fontProfile}.monospace.package -> download monospace for this font
│   │   └── ${fontProfile}.regular.package -> download regular for this font
│   └── fonts.fontconfig -> allow custom font profiles
├── monitors -> add custom monitors
└── wallpaper -> select custom monitors
```
